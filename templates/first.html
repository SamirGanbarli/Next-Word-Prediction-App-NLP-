Development of Language Model for Azerbaijani Language

Development

By utilizing an n-gram model using some algorithms and built-in functions in Python NLTK, the program takes an input (one word or two words) from the user and gives the most possible next word in the output. The project is mainly divided into two parts – sentence generator and bigram, trigram generators. Core of the program is frequency distribution algorithms which has many useful built-in functions.

Design

Using NLTK, the program first generates the n-grams – mainly bigrams. After that point, frequency distribution algorithms calculate important statistics and matches the input with data corpuses to find the most possible next word.

Creativity

For instance, we have an idea about creating a database that consist of small texts on various types such as poetry, narrative or personal response. For this idea, we need to create a large database from scratch, and this can take a lot of time. Thus, when the system receives a sentence from the user, it can generate small relevant text based on a specific input.
